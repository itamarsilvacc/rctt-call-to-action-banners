Esse plugin possibilita que o usuário adicione banners com link e imagem que possam ser inseridos em qualquer post utilizando shortcodes.

== Installation ==
1) Faça o upload do plugin em Plugins > Adicionar novo > Enviar plugin e clique em \"Instalar agora\".

2) Ative o plugin em Plugins > Todos os plugins.

3) Crie os banners na nova opção de post disponível no painel, CTA\'s.

4) Exiba um banner usando o shortcode [banner-cta id=\"id_post\"].
